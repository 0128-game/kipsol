# kipsol
このメモ帳では数字は対応していません。


使える文字は

i,u,o,t,k,s,p,d,g,z,b,l,n,カンマ,ピリオドのみ。

文字は子音にのみ存在する。

文字は基本中心に点がある円とその円の中身で構成される。

また、文字には大小が存在し、有声音には大きな円が、無声音には小さな円が割り振られており、有声音と無声音は対応している。ただし、l,nは大きい円＋小さい円の二重円と円の中身で表現され、小さい円は存在しないものとする。

それぞれの円の中身

z,s:なし

d,t:中心から伸びる3等分線

g,k:中心から伸びる4等分線

b,p:中心から伸びる6等分線

l:二重円のみ

n:二重円の中心から伸びる3等分線

また、カンマは語末記号代わりとして利用する。

語末記号は小さい円よりもさらに小さい点のような円で表現する。

また、ピリオドは文末記号として利用し、文末記号はは小さい円の半径ほどの長さの縦の直線を二本横に並べた記号である。

母音は子音同士の接続方法で表す。

右方向にまっすぐ接続するとo

右上に30度傾けて接続するとi

右下に30度傾けて接続するとu

になる。

ただし、子音が連続する場合、右方向にまっすぐ文字の中心同士を直線でつなぐ。

また、単語の最後が母音の場合、語末記号を子音にそのままくっつける。

向きは通常と同じ。

単語の最後が母音の場合は、子音の連続と同じように子音と語末記号を直線でつなげる。

また、文末には、語末記号と文末記号の両方を使用する。

この言語では、母音から始まる単語、母音の連続は存在しない。
